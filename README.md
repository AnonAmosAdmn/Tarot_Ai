===============================================================

# Download the Game:

git clone (https://github.com/AnonAmosAdmn/Tarot_Ai.git)

===============================================================

# Create a Virtual Environment:

python -m venv mygame_env

source mygame_env/bin/activate  # Linux/macOS

mygame_env\Scripts\activate     # Windows

===============================================================

# Edit .env file:

input your api key

===============================================================

# Then install Pygame:

pip install pygame

===============================================================

# Run the game :

python3 tarot.py

===============================================================
